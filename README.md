
## Data Vault DBT Project for Streaming Activity Dataset

This project aims to implement a Data Vault methodology using the Data Build Tool (DBT) for processing and analyzing streaming activity data. The project is designed to work seamlessly with Snowflake and Redshift databases using SQL scripts.

## Key Technologies

DBT (Data Build Tool): Used for transformation and modeling of data in the Data Vault architecture.

Snowflake: Cloud-based data warehousing platform.

SQL

Redshift: Data warehouse service in the cloud.
## Installation

Clone this repository to your local machine.

Set up your DBT profiles by modifying the profiles.yml file with your database connection details.

    

## Skills

SQL, Python, Redshift, Snowflake
